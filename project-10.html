AGGREGATOR OF AGGREGATORS

Data aggregators have taken over the world completely. From Yelp to FoodPanda, Expedia to TripAdvisor, platforms have emerged which has changed the way data is consumed and presented. However with the emergence of such data-rich platforms it becomes extremely necessary to vet and compare these platforms to understand and compare their correctness and find unconcious biases if any.

As a part of the project we tried to aggregate data from three different aggregators namely Trip Advisor, Yelp and Zomato. Once the data was accessed we cleaned the data, and then aggregated the data for quantitative as well as qualitative comparison's. Below are some of the important details of the project:

Data Sources:

  • ZOMATO - ZOMATO API
  • YELP - YELP FUSION API
  • TRIP ADVISOR - KAGGLE

Metrics for Comparison

  • PRICE DIFFERENCES
  • RATING DIFFERENCES

Technologies Used:

  • AWS EC2
  • DOCKER CLI
  • GOOGLE CLOUD PLATFORM
  • AZURE
  • KUBERNETES
